RAG-ответы по базе знаний

Разработайте полноценный pipeline RAG-системы (Retrieval-Augmented Generation), которая по входному пользовательскому вопросу находит релевантные фрагменты в предоставленном корпусе данных и формирует на их основе качественный ответ.

Для решения задачи вам предоставляется расширенный набор данных в формате CSV:

questions.csv – база пользовательских запросов, для которых необходимо выстроить процесс поиска (retrieval) и генерации ответа;
websites.csv – корпус знаний, использующийся в качестве источника информации для retrieval-части пайплайна;
sample_submission.csv – пример файла решения для загрузки на платформу.

Ваша цель — создать систему, которая эффективно извлекает нужный контекст из базы знаний и генерирует точный ответ, соответствующий запросу пользователя.
