# Кредитный скоринг

Разработайте модель прогнозирования выхода клиента в дефолт на основании истории карточных транзакций. Модель должна качественно отделять потенциальных клиентов-неплательщиков от тех, кто способен своевременно выплачивать полученный займ.

Для решения задачи вам предоставляются следующие данные:

1. train_data.parquet – тренировочный датасет с историей транзакций;
2. test_data.parquet – тестовый датасет для формирования прогнозов;
3. train_target.csv – файл с целевыми метками (классами дефолта) для тренировочной выборки.

Полный набор данных был разделен на несколько частей одинаковой структуры для удобства обработки. Ваша задача — обучить модель на тренировочных данных и подготовить файл с предсказаниями для тестового набора.
